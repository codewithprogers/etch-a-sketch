# etch-a-sketch
A browser-based drawing pad (something between a sketchpad and an Etch-a-Sketch) built as part of  The Odin Project course.

## Features
- Interactive drawing grid - Click and drag to draw.
- Custom grid sizes - Choose how many squares (e.g. 16x16, 32x32, etc.).
- Color Modes
- Clear board - Reset your canvas instantly.
- Responsive layout - Works on different screen sizes.

## Technologies Used
- HTML - Structure
- CSS - Layout and styling
- JavaScript - Dynamic grid creation and user interaction
